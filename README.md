## Install

Via Composer

``` bash
$ composer require renderbit/light-bootstrap-dashboard
```

## Usage

``` php
$skeleton = new RenderbitTechnologies\LightBootstrapDashboard();
echo $skeleton->echoPhrase('Hello, League!');
```
